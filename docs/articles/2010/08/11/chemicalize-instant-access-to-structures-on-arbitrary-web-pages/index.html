Chemicalize: Instant Access to Structures on Arbitrary Web Pages

By Richard L. Apodaca

2010-08-11T00:00:00.000Z

The Problem: A Web page has many chemical names, but no structures.

The Solution: Paste the page's URL into Chemicalize. The page looks the same, but when you hover over a substance name, you get a popup showing the structure (and a link to a structure summary page).

Pretty cool, huh?
