Latest CTFile Formats Specification Available Now from Symyx

By Richard L. Apodaca

2010-06-28T00:00:00.000Z

The latest version of the CTFile format specification is now available for download. This version, printed in June 2010, covers both the widely-used V2000 molfile format and the newer V3000 format.

The specification references no fewer than six times another document titled Symyx Chemical Representation, which apparently contains important information for implementing molfile readers/writers. Unfortunately, I have not been able to locate it anywhere.
