ChemWriter 1.3.1

By Richard L. Apodaca

2008-06-30T00:00:00.000Z

ChemWriter 1.3.1 can now be downloaded. This version resolves an [EditorApplet](http://metamolecular.com/chemwriter/draw/) issue in which molfiles containing two or more atoms with exactly the same 2D coordinates were not displayed properly.

Details are available on the Metamolecular Company Blog.

ChemWriter is the 2D chemical structure editor designed for Web applications. Lightweight and intuitive, ChemWriter makes an excellent choice for both creating and displaying 2D chemical structures on the Web.
